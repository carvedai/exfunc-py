# GoogleSearchJobPostingsDatePosted

Filter for job postings based on when they were posted


## Values

| Name            | Value           |
| --------------- | --------------- |
| `ANY_TIME`      | Any time        |
| `PAST_24_HOURS` | Past 24 hours   |
| `PAST_3_DAYS`   | Past 3 days     |
| `PAST_WEEK`     | Past Week       |
| `PAST_MONTH`    | Past Month      |
